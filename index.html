Introducing WebCBT

28 Dec 2014

WebCBT is a free and open source web based CBT - Cognitive Behavioral Therapy tool to manage your thoughts, feelings and behaviours using CBT.

First version of WebCBT was released on 13th January 2015 and is available for Download.
